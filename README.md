## Банковские операции

Набор инструментов для обработки банковских операций, а именно:
- маскирование номеров карт и счетов; 
- фильтрацию и сортировку операций по различным критериям.

## Функционал

### Маскировка данных

Автоматически определяет тип введенных данных (карта или счет) и применяет соответствующую маскировку.

1. **Маскировка номеров карт**
   - Принимает на вход строки и числа
   - Выводит строку номера в формате: `XXXX XX** **** XXXX`
   - Проверяет корректность номера (16 цифр)

2. **Маскировка номеров счетов**
   - Принимает на вход строки и числа
   - Выводит строку с 4 последними цифрами: `**XXXX`
   - Проверяет корректность номера (20 цифр)

### Обработка дат

- Преобразование даты из формата ISO (`2024-03-11T02:26:18.671407`) в формат `ДД.ММ.ГГГГ`

### Фильтрация и сортировка операций

1. **Фильтрация по статусу**
   - По умолчанию фильтрует выполненные операции (`EXECUTED`)
   - Поддерживает любой другой статус операции (например, `CANCELED`) 
   
2. **Сортировка по дате**
   - По умолчанию сортирует по убыванию (новые сначала)
   - Поддерживает сортировку по возрастанию
     
## Тестирование

**В проекте используются `pytest` и `pytest-cov`.**

**Тесты находятся в папке `tests`** 
   - `tests/test_masks.py` — тесты функций маскировки карт и счетов.
   - `tests/test_widget.py` — тесты функций обработки строк с картами, счетами и датами.
   - `tests/test_processing.py` — тесты фильтрации и сортировки банковских операций.

**Запуск всех тестов**
   - poetry run pytest

**Запуск тестов с отчетом покрытия**
   - poetry run pytest --cov=src --cov-report=term-missing --cov-report=html:coverage_html_report

**HTML-отчет покрытия создается в папке coverage_html_report. Текущее покрытие функционального кода: 100%.**
